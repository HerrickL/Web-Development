How-To: NBA Stats API

Connect with and use the NBA Stats API

Introduction

In 2013 the NBA made their stats database free for the public to access. This database includes both current and past season stats for all players, teams, and games in league's 69-year history. Previously, fans and all but a select few media outlets had no comprehensive database to reference, and typically relied on their own notes to keep track of stats for any comparisons not provided by the league. This recent access has been an exciting and monumental development for anyone from relative newcomers to die-hard fans who want to throw their voice into the online basketball conversation.


Although open and ready for use, at this time there is still no official (and very sparse unofficial) documentation for access and use of the NBA stats API. What little documentation does exist is either unfinished or hidden behind paywalls. This how-to goes over how to prepare to connect to the API, avoid cross-domain request errors, access the data received, and put that data to use.

Prior To This How-To

This documentation requires you to have basic knowledge in HTML, JavaScript, jQuery, and some previous experience working with APIs. It is helpful but not required to have a basic understanding of NBA basketball terms, players, and teams. Before getting started, it is recommended to take a few minutes to become familiar with stats.nba.com as that is the point of reference for this guide.
