How-To: NBA Stats API

Connect with and use the NBA Stats API

Understanding the URLs

There are multiple endpoint URLs used in making a query to the NBA API. Since there is no official documentation, it isn't perfectly apparent just how many endpoints there are. For the purpose of this documentation, I have included endpoints identified through a project by Nick Bottomley, who created a Node.js wrapper around the NBA's API. I have further provided all of the needed URL parameters through testing each endpoint and checking for errors.


Player Profile

url: http://stats.nba.com/stats/playerprofile
Needed Parameters: LeagueID, PlayerID, Season, SeasonType, GraphStartSeason, GraphEndSeason, GraphStat


Common Info About a Player

url: http://stats.nba.com/stats/commonplayerinfo
Needed Parameters: Player ID


Team Data by General Splits

url: http://stats.nba.com/stats/teamdashboardbygeneralsplits
Needed Parameters: SeasonType, TeamID, MeasureType, PerMode, PlusMinus, PaceAdjust, Rank, Season, Outcome, Location, Month, SeasonSegment, DataFrom, DateTo, OpponentTeamID, CsConference, VsDivision, GameSegment, Period, LastNGames


Common Player Data by General Splits

url: http://stats.nba.com/stats/playerdashboardbygeneralsplits
Needed Parameters: MeasureType, PerMode, PlusMinus, PaceAdjust, Rank, Season, SeasonType, PlayerID, Outcome, Location, Month, SeasonSEgment, DateFrom, DateTo, OpponentTeamID, VsConference, VsDivision, GameSegment, Period, LastNGames


Shot Chart

url: http://stats.nba.com/stats/shotchartdetail
Needed Parameters: SeasonType, TeamID, PlayerID, GameID, Outcome, Location, Month, SeasonSegment, DateFrom, DateTo, OpponentTeamID, VsConference, VsDivision, Position, RookieYear, GameSegment, Period, LastNGames, ContextMeasure


Scoreboard

url: http://stats.nba.com/stats/scoreboard/
Needed Parameters: GameDate, LeagueID, DayOffset


Play by Play of Game

url: http://stats.nba.com/stats/playbyplay
Needed Parameters: GameID, StartPeriod, EndPeriod


Boxscore of Game

url: http://stats.nba.com/stats/boxscorescoring
Needed Parameters:GameID, StartPeriod, EndPeriod, StartRange, EndRange, RangeType
